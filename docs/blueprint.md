# **App Name**: SRM Study Buddy

## Core Features:

- Secure User Authentication: Robust user registration, login, and session management using JWT, bcrypt for password hashing, and role-based access for students and teachers.
- Study Session Tracker: Students can start, stop, and log their study sessions by subject, with basic analytics showing study hours over time.
- AI Communication Practice: Students can record audio, which is then processed, transcribed, and sent to a Google Generative AI tool that provides feedback on clarity, grammar, and overall communication skills.
- Interactive Coding Practice: Display coding problems within an in-browser editor, allow code submission, and provide validation of answers and store submission history.
- Dynamic Quiz System: Students can take quizzes with multiple-choice questions, receive instant scoring, and track their performance results.
- Teacher Performance Dashboard: Teachers can view analytics for student study time, quiz results, coding performance, and communication feedback scores across their students.

## Style Guidelines:

- The visual scheme embraces a light, professional aesthetic. The primary color is a composed, academic blue (#2966A3), chosen for its ability to convey focus and reliability without being overly aggressive. It complements the platform's educational purpose and is suitable for prominent UI elements like headers and interactive components.
- The background color is a subtle, almost imperceptible light blue-grey (#F1F5F8), maintaining a clean canvas that supports content readability and reduces eye strain during extended study sessions. Its heavily desaturated nature ensures it remains in the background.
- An energetic accent color, a vibrant teal (#26DECC), is used for highlights, calls-to-action, and interactive elements to provide visual emphasis and a touch of modern dynamism. Its hue is analogous to the primary blue, creating a harmonious yet distinct contrast in brightness and saturation.
- For headlines and emphasized text, the sans-serif font 'Space Grotesk' provides a modern, slightly technical feel, aligning with a tech-focused learning platform. For body text, the neutral and highly readable sans-serif 'Inter' is selected to ensure clarity across all content types.
- For all code snippets and programming environments, the monospace font 'Source Code Pro' will be used, offering optimal readability and proper alignment for programming constructs.
- Use a set of clean, simple line icons that clearly communicate their function. Icons should be easily understandable, promoting intuitive navigation and user experience within an educational context.
- Implement a clear, well-structured layout with ample whitespace to enhance readability and reduce cognitive load. Dashboards should be designed with modular components for easy performance overview, while content pages should prioritize focused engagement.
- Incorporate subtle and smooth animations for transitions between sections, state changes (e.g., loading indicators, form submissions), and interactive elements to provide positive user feedback without distracting from the learning content.
